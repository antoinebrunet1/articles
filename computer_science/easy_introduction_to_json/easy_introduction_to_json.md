# Easy introduction to JSON

**Author:** Antoine Brunet

**Created:** March 6, 2026

JSON is a text format of any of those types:

- `string`
- `number`
- `true`
- `false`
- `null`
- `object`
- `array`

## `string`

Syntax: `"Some text of 0 characters or more"`

## `number`

Examples:

- `100`
- `-100`
- `100.25`
- `-100.25`

## `true`

Syntax: `true`

## `false`

Syntax: `false`

## `null`

Syntax: `null`

## `object`

Syntax:

```
{
    "nameOfValue1": value1,
    "nameOfValue2": value2,
    ...
}
```

- 0+ named values
- Every value is of any type.
- Unordered named values

## `array`

Syntax :

```
[
    value1,
    value2,
    ...
]
```

- 0+ values
- Every value is of any type.
- Ordered values
