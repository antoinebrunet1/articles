# Non-AI automatic code generation

**Author:** Antoine Brunet

**Created:** March 21, 2026

In this article, I will present non-AI automatic code generation possibilities I actually used for two tools.

## 1. IntelliJ

### What is IntelliJ?

IntelliJ is a software that can be used to write code. I use it to code in Java.

### Non-AI automatic code generation

I have used the getters and setters automatic generation for classes in IntelliJ while doing back-end development. It is easy to access with a right click inside the class. You can have a look at [the tutorial about that](https://www.jetbrains.com/help/idea/generating-code.html#generate-getters-setters) for more information.

## 2. Playwright (Node.js version)

### What is Playwright?

Playwright is a framework to test web apps and APIs.

### Non-AI automatic code generation

The `codegen` command opens up a browser and a code window. In the browser, you can for example click on something and the code for that clicking action will appear in the code window. You can copy the code from the code window to add it to your code.
